1. Installation Instructions

Attention! Do not rename Rise of Mankind folder or the mod won't work properly. If you are installing full mod make sure that previous Rise of Mankind version is deleted from Mods folder first. If you are installing patch do not delete the old version from Mods folder.

  1. Locate your Civilization 4 installation folder from your hard drive.
  2. There go to subfolder named Beyond the Sword and under Beyond the Sword go to subfolder Mods.
  3. Unzip the mod to this Mods folder. Zip file contains the needed folder structure for the mod files so make sure your archiver program is set to unzip files with the folders.
  4. Start your Civ 4: Beyond the Sword normally and from start menu click "Load a mod". Choose Rise of Mankind mod from list and click ok.
  5. When game has reloaded itself with the mod, you can start playing.

Saved games from previous mod versions do not work after patching the mod!

3. About the mod

The Goal

This mod is my attempt to enhance Civilization 4 to a whole new level. This completely free "Expansion pack" keeps the basic Beyond the Sword game mechanics but converts them to much more complex system which makes this modpack to feel like it's completely new game.

Ever since Civilization 1 I've liked this series a lot but during Civ 3 I found mods Double Your Pleasure and bit later Rise and Rule which made the game be so much more fun than before that when finally Civ 4 was released, I felt disappointed as it didn't offer as much as did DYP or RAR. Civ 4 had some new elements like religion and civic system (like in Alpha Centauri) but I wanted more techs, more units, more complexicity so I began working on my own mod. I started by going through most of the popular mods and check what's been done already and then by using those mods loosely as my guide, I created this mod.

The Concept

This mod is specifically designed for myself and I continue to make changes to the direction that I feel like is correct and which make me like more this game. Aim of the mod is to be more accurate historically which is why the whole tech tree has been redone and many new techs has been added. New techs meant much more room for new buildings and for new units and adding those was my next step. Once I had added most techs, units and buildings it came clear to me that I need to redo unit categories, promotions and change many unit abilities so that more modern units are really superior to older unit types. Note that now you must consider more carefully what to build in city because the number of different buildings has increased so much and because you are now limited to 4 World Wonders per city. I used most of my freetime past three years to make this mod so there are numerous changes compared to original game but I didn't keep record of every little change - you'll have to check Civilopedia entries for all the changes. Players' feedback will help me improve this mod and find the correct balance for everything so please post to Rise of Mankind forums if you notice things in the mod that need some tweaking.

Features

Because the modpack contains now numerous other mods the feature amount has grown steadily over the years. The first thing you'll notice when you start a new game is that there's new map sizes and new game speeds. Giant and Gigantic map sizes give more room for larger empires and for more players but the downside is that these map sizes require quite a lot from your computer so you should only try these maps if you have high-end computer, preferably with 64bit operating system.

When the game starts you'll notice the next improvement which is the interface changes. In Rise of Mankind the interface is fully customizeable through BUG options which you can access by clicking BUG button on top left of the screen or hitting CTRL+ALT+O. The options have been divided into different pages and if you hover your mouse pointer over any option, you'll see help tips for them. Experiment with the settings and change the game look to your own liking.

In Custom Game menu if you scroll down the options list you'll see that there's several new game options. Most of these options are on by default: Revolutions, Inquisitions, Barbarian Civs, Dynamic Civ Names, Tech Diffusion. Rising Seas and Barbarian World options are off by default.

Transhuman Era

Transhuman Era is a new game era between Modern Era and the Future tech. The new era contains many present day and near future techs which give access to the most powerful mech, robot and nanite units and to all kinds of new buildings and wonders. In making the new Transhuman era I used Galactic Civilization and Alpha Centauri games as guides so if you've played those games you'll notices similarities. To keep it bit more realistic, I've gone through dozens of science articles and science websites to look info about near future techs. Many of the new techs are based on real life research and many transhuman era units are based on prototypes that exists today in our world.

Game length

I recommend playing this mod on Marathon game speed and on huge map size. I prefer making custom game and use Smartmap mapscript for making good continental maps. Mod has also pre-made huge Earth map scenario with 28 Civilizations which all are on their historical starting locations. If you prefer extra long game you could try out the new Snail game speed which has 3000 turns.

Translations

The mod works best on English language setting. Germany, Spanish, French and Italian work as well but translations for those are incomplete. For other languages the translation has just begun but there are other modders who have already translated this modpack to various languages like Polish, Russian, Chinese and Japanese.

5. Mod settings

Rise of Mankind modpack has many individual mods included. Few of the major mods are RevolutionDCM and BUG. Because the mods have been made by various modders in www.civfanatics.com, most of mods use different methods to change mod settings:

BUG mod v4.3

- in game press CTRL+ALT+O to open up BUG mod option screens. BUG mod saves your settings after you close the option screen so you don't need to change the settings every time you launch the mod.

- If you want to use Advanced Unit Naming mod feature from BUG mod, you'll have to edit Adv Unit Naming.ini in Rise of Mankind -folder.

Enhanced Tech Conquest

- open Rise of Mankind config.ini to notepad. Each value has description above it so it should be easy to change the values.

Revolution

- Can be turned off from Game Options when starting Custom Game (scroll down the option list)

- Some of the settings can be found from Revolution.ini in the Rise of Mankind\UserSettings -folder

- More settings in RevolutionDCM.ini -file after you've made one game with Revolution components on

Dale's Combat Mod

- in game press CTRL+ALT+O and open RevDCM options page

Influence Driven War

- in game press CTRL+ALT+O and open RevDCM options page

Inquisition

- Can be turned off from Game Options when starting Custom Game (scroll down the option list)

Limited Religions

- in game press CTRL+ALT+O and open RevDCM options page

Super Spies

- in game press CTRL+ALT+O and open RevDCM options page

Dynamic Civ Names

- Can be turned off from Game Options when starting Custom Game (scroll down the option list)

Tech Diffusion

- Can be turned off from Game Options when starting Custom Game (scroll down the option list)

Barbarian Civs

- Can be turned off from Game Options when starting Custom Game (scroll down the option list)

Start as Minor Civs

- Can be turned on from Game Options when starting Custom Game (scroll down the option list)

Barbarian World

- Can be turned on from Game Options when starting Custom Game (scroll down the option list)

Rise of Mankind

- Edit GlobalDefines.xml and GlobalDefinesAlt.xml -files in the Rise of Mankind\Assets\XML -folder

6. MODULES (optional)

  • Holy Wars
    • Off by default, can be enabled by moving Holy Wars folder with its contents from Unloaded Modules folder to Modules folder
    • This module enhances currently only Christianity's medieval era units and wonders (might be later expanded to enhance all religions) by adding many "crusader" unit types with specific abilities.
  • No Building Graphics
    • Off by default, can be enabled by moving NoBuldingGraphics folder from Unloaded Modules\Custom Buildings folder to Modules\Custom Buildings folder
    • This module disables all regular building graphics from the mod and thus reduces the required amount of graphics memory.
  • Static Leaderheads
    • Off by default, can be enabled by moving Static folder from Unloaded Modules folder to Modules folder
    • This module disables animations from all default leaderheads that were included in Beyond the Sword expansion pack. This module does not affect custom leaderheads (support might be added later).
    • The module speeds up diplomacy as the game doesn't have to load all animation files and this also reduces the required amount of graphics memory.
  • Realistic Diplomacy
    • Off by default, can be enabled by moving Realistic Diplomacy folder with its contents from Unloaded Modules folder to Modules folder
    • This module changes diplomacy by making AI players forget past events and wars faster, thus diplomacy is much more dynamic and it can change quite rapidly.

See also RoM Concepts pedia pages in game for more specific info what each mod feature does.

8. Known issues

- on some operating system configurations the user may suffer from 'no interface' -problem. For this problem the mod has file named 'CvAltroot.py' in 'Rise of Mankind' folder. If you are experiencing this problem, open this file to Notepad and follow the instructions there. This should fix the problem. - Civilization modules for regular BtS might not work correctly with this mod due to civic changes in the mod

- unit modules for regular BtS might not work correctly with this mod due to unitcombat type changes in the mod

- When playing on huge or bigger maps, in late game you may encounter Memory Allocation Faults (MAFs), these happen due to bug in game engine, not bug in the mod. Best way to avoid them is to get more than 2GB RAM and get a graphics card with at least 512MB video RAM. You can also enable Static Leaderheads module and disable civ specific unit artstyle modules (see RoM Concepts pedia for more info).

- Giant or Gigantic map sizes might not work correctly on all mapscripts, I recommend making maps for these sizes with advanced mapscripts like Smartmap

- Some mapscripts can not (yet) generate Marsh terrain, these will just generate the basic type terrains (same as in Beyond the Sword)
